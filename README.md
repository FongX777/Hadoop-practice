Massive Data Analysis on Hadoop
===========================

Here are some projects I've written just for practice.

The project list: 
	* Matrix Multiplication
	* Page Rank
	* Kmeans


In every project, there are 
	* README.md  
	* Spec.pdf
		describe the problem
	* Code.java  
		the java code
	* Report.pdf
		explain the project and how to implement it 
  * input.txt
		the input of data

	* Output data.txt
		the output of data
